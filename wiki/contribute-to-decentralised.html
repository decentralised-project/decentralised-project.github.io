Contribute to Decentralised_

Everything related to Decentralised, even the website is completely open source, so anyone can get involved.

Getting started_

  • Register for a Github account at https://github.com/join, and log in.
  • Get a Git client for your operating system. (eg. https://www.sourcetreeapp.com/)

Getting the source_

  • Find the DC repository you're interested in on Github, and click the FORK button to make a copy in your own account.
    • https://github.com/decentralised-project/dc-viewer 3D Viewer
    • https://github.com/decentralised-project/dc-console Console/Terminal Client
    • https://github.com/decentralised-project/dc-core Core API
    • https://github.com/decentralised-project/decentralised-project.github.io This Website
  • Using your Git client, CLONE the repository in YOUR Github account to your local machine.
    For example, the source could be https://github.com/[myUsername]/dc-viewer and destination c:\Projects\dc-viewer

Saving your changes_

When you've finished making changes to your local working copy, you need to save them back to your Github account.

  • Using your Git client, COMMIT your changes.
  • Again, using your Git client, PUSH your commit, to upload the changes to Github.

Submitting your changes for review_

When you're happy with the changes you've made, you'll want to merge them into the main Decentralised project so they're included in the next release, or shown on the website.

  • Log into Github, and navigate your web browser to the repository in your account.
  • Click Create Pull Request
  • Fill in the short form and submit.

Your changes will be reviewed and you'll receive feedback. If there are no problems with your changes, they'll be merged into the main projects by an administrator.
