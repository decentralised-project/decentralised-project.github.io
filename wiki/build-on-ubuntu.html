Build Decentralised on ubuntu

These build instructions were tested as they were written on a clean installation of ubuntu 16.04 LTS 64-bit running on VirtualBox.

Tip: Before you start setting up your build environment, check you're running 64-bit ubuntu if you're planning on building for Android later.

Preparing the environment

  • Open a terminal and type ... 
    sudo apt update
  • If you're running ubuntu in VirtualBox, in the Devices menu at the top of the screen, select "Insert Guest Additions CD image..." to install the plugin, and enable clipboard sharing etc. Then restart VM.
  • Install the build tools, type ... 
    sudo apt-get install build-essential automake libtool cmake-qt-gui git mercurial
  • Install the dependencies, type ... 
    sudo apt-get install libfreetype6-dev libfreeimage-dev libzzip-dev libxrandr-dev libxaw7-dev freeglut3-dev libglew-dev libgl1-mesa-dev libglu1-mesa-dev libpoco-dev libboost-all-dev

Build/Install Ogre

  • Open a terminal and type ... 
    mkdir Decentralised
cd Decentralised
hg clone https://bitbucket.org/sinbad/ogre ogre -r v1-9
mkdir ogre-build
cmake-gui
